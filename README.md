# TODO LIST Project

## 인프런 스터디 클럽 미션 1단계 프로젝트

- NextJs, Supabase 를 이용한 TODO List 페이지 구현

### 사용기술

---

- NextJS 14^
- React 18^
- React Query
- Material TailwindCss 2.1.10
- Supabase

---
